#  My-App

**My-App** is a modern web application built with Vue.js, featuring user management, storage management, and authentication.

##  Project Setup

### 1️⃣ Install Dependencies
```sh
npm install
```
### 2️⃣ Run in Development Mode
```sh
npm run serve
```
